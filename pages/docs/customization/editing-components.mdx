### Customizing Existing Components
To do this we can navigate to the `components` folder and make changes to any of the starter components. Let's take a look at a quick example:
#### Customizing the `Header` Component
Here is the default `Header` component:
<br/>
<TitleCode title="components/Header.js">
```jsx
export default function Header(){
  return (
    <div>
      <div className="fixed bg-white top-0 left-0 p-8 w-full z-50 border-b">
        <div className="font-bold">
          ðŸ”¥ Welcome to Bon5R!
        </div>
      </div>
      <div className="pb-[150px]">
      </div>
    </div>
  );
}
```
</TitleCode>
<br/>
Let's say we want to have a different title for each page, (for instance the title of the a blog post).
Using the `props` parameter we can create a input for the user so that for every usage
the component would reaquire a title.
<br/>
<TitleCode title="components/Header.js">
```jsx
export default function Header(props){
  return (
    <div>
      <div className="fixed bg-white top-0 left-0 p-8 w-full z-50 border-b">
        <div className="font-bold">
          {props.title}
        </div>
      </div>
      <div className="pb-[150px]">
      </div>
    </div>
  );
}
```
</TitleCode>

### Usage
Now the user can use the customized component in pages like so:

```md

<Header title="ðŸ”¥ My Custom Header" />
```
