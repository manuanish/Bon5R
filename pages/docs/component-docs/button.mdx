  # Button
  <Contributors contributors={["manuanish"]} readTime={2} />
  ---
  The Button component can be used to as a link, like counter, etc.
  <br/>
  ## Usage
  We can use the Button component like so:
  <TitleCode title="button.mdx">
  ```jsx
  import Button from "@components/Button";

  <Button
    href="#"
    text="My MDX Button"
  />
  ```
  </TitleCode>
  <br/>
  <Alert type="warning">The `href` prop should not be left empty. This may cause errors.</Alert>
  <br/>
  This we be rendered as shown:
  <Demo>
    <Button
      href="#"
      text="My MDX Button"
    />
  </Demo>
  <br/>

{" "}

<br />
### Props

| Name   | Type     | Default | Description                                                       |
| ------ | -------- | ------- | ----------------------------------------------------------------- |
| `href` | `string` | `""`    | The page the button links to. <br />**Should not be left empty!** |
| `text` | `string` | `""`    | The inner text of the button.                                     |
