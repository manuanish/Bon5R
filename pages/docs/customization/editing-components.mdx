  # Editing Built in components
  <Contributors contributors={["manuanish"]} readTime={5} />
  ---
  In this page we will look at an example that describes how we might customize a Bon5R component.
  <br/>
  ## Customizing the `Footer` component
  Let's begin by locating where the source code for the `Footer` component is located. If we navigate to the `components`
  folder we can see that all the built-in components are stored there. Let's take a look at the `components/Footer.js`
  <br/>
  <TitleCode title='Footer.js'>
  ```jsx
  export default function Footer(props) {
    return (
      <div className="text-center mb-5 mt-5 text-neutral-300">
        {props.children}
      </div>
    );
  }
````
<br/>
</TitleCode>
From here we can change the code and make it fit our theme/needs. In this example I have made it a link to a GitHub account
### Modified `Author` component
<TitleCode title="Author.js*">
```jsx
import Link from "next/link";

export default function Footer(props) {
  return (
    <Link href={`https://github.com/${props.userName}`}>
      <div className="text-center mb-5 mt-5 text-neutral-300">
        {props.children}
      </div>
    </Link>
  );
}
```

  </TitleCode>
  We will now look at how we might use this modified component in our application
  ### Usage
  We can import the component exactly as before and it should work.
  <TitleCode title="author-modified-demo.mdx">
  ```jsx
  import Footer from "@components/Footer";

  <Footer userName="manuanish">Code with ❤️ by @manuanish</Footer>
  ```
  </TitleCode>
  <br/>
  This should give us the following
  <ToggleDemo title="FooterModified Demo" childrenLight={
    <FooterModified userName="manuanish">Code with ❤️ by @manuanish</FooterModified>
  } childrenDark={
    <FooterModified userName="manuanish">Code with ❤️ by @manuanish</FooterModified>
  } />
  <br/>
  ### Props

| Name           | Type      | Default | Description                                                           |
| -------------- | --------- | ------- | --------------------------------------------------------------------- |
| `userName` | `string`    | `""`    | Desired GitHub username. |
