### Customizing Existing Components
To do this we can navigate to the `components` folder and make changes to any of the starter components. Let's take a look at a quick example:
#### Customizing the `Header` Component
Here is the default `Header` component:
<br/>
```jsx
import * as React from 'react';

function Header(){
  return (
    <div>
      <div className="fixed bg-white top-0 left-0 p-8 w-full z-50 border-b">
        <div className="font-bold">
          ðŸ”¥ Welcome to Bon5R!
        </div>
      </div>
      <div className="pb-[150px]">
      </div>
    </div>
  );
}

export default Header;
```
<br/>
Let's say we wanted allow the user to have a different title for every page. Using the `props` input we can create a input for the user so that for every call
we would require a input to have a header.
<br/>
```jsx
import * as React from 'react';

function Header(props){
  return (
    <div>
      <div className="fixed bg-white top-0 left-0 p-8 w-full z-50 border-b">
        <div className="font-bold">
          {props.title}
        </div>
      </div>
      <div className="pb-[150px]">
      </div>
    </div>
  );
}

export default Header;
```

Now the user can use the component in pages like so:

```md
import Header from '@components/Header';

<Header title="ðŸ”¥ My Custom Header"/>
```
