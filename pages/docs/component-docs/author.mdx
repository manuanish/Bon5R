  # Author
  <Contributors contributors={["manuanish"]} readTime={2} />
  ---
  The Author component can be used to preface a page linking to the author's GitHub account.
  <br/>
  ## Usage
  We can use the Author component like so:
  <TitleCode title="author.mdx">
  ```jsx
  import Author from "@components/Author";

  <Author
    userName="manuanish"
    readTime={5}
    date="3/14/15"
  />
  ```
  </TitleCode>
  <br/>
  This we be rendered as shown:
  <ToggleDemo padding={"10"} title="Author Demo" childrenLight={<Author
    userName="manuanish"
    readTime={5}
    date="3/14/15"
  />} childrenDark={<Author
    userName="manuanish"
    readTime={5}
    date="3/14/15"
  />}/>
  <br />
  ### Props

| Name       | Type      | Default | Description                                               |
| ---------- | --------- | ------- | --------------------------------------------------------- |
| `readTime` | `integer` | `{}`    | This is the the approximated reading time of the article. |
| `userName` | `string`  | `""`    | This is the GitHub username of the author.                |
| `date`     | `string`  | `""`    | This is the date of posting.                              |
