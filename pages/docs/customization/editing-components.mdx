  # Editing Built in components
  <Contributors contributors={["manuanish"]} readTime={5} />

  In this page we will look at an example that describes how we might customize a Bon5R component.
  <br/>
  ## Customizing the `Author` component
  Let's begin by locating where the source code for the `Author` component is located. If we navigate to the `components`
  folder we can see that all the built-in components are stored there. Let's take a look at the `components/Author.js`
  <br/>
  <TitleCode title='Author.js'>
  ```jsx
  import * as React from "react";
  import Image from "next/image";

  function Author(props) {
    return (
      <div>
        <div className="flex text-[gray]">
          <div className="mt-[4px]">
            <Image
              alt="avatar"
              width="20"
              height="20"
              className=" rounded-lg"
              src={"https://github.com/" + props.displayName + ".png"}
            />
          </div>
          &#8198;&#8198;&#8198;
          <a
            href={"https://github.com/" + props.displayName}
            target="_blank"
            rel="noreferrer"
          >
            <strong>
              <span>{props.author}</span>
            </strong>
          </a>
          &#8198;&#8198;â€”&#8198;&#8198;<span>{props.date}</span>
        </div>
      </div>
    );
  }

  export default Author;
  ```
  <br/>
  </TitleCode>
  From here we can change the code and make it fit our theme/needs. In this example I have decided to include a way to "map" authors into the component so that more than one author can write a page.

  ### Modified `Author` component
  <TitleCode title="Author.js*">
  ```jsx
  import Image from "next/image";
  import Link from "next/link";

  export default function Author(props){
    return (
      <div className="flex gap-1">
      <span className="text-slate-500 text-lg">Written by&nbsp;</span>
        {props.contributors.map((userName) => (
          <div className="cursor-pointer rounded-full border-2 w-[30px] h-[30px] border-slate-500 hover:border-slate-400">
            <Link href={`https://github.com/${userName}`} passHref>
                <Image
                  alt={userName}
                  width="30"
                  height="30"
                  className="rounded-full"
                  src={`https://github.com/${userName}.png`}
                />
            </Link>
          </div>
        ))}
        &nbsp;<span className="text-slate-700 text-lg">&#8212;&nbsp; {props.readTime} min read</span>
      </div>
    );
  }
  ```
  </TitleCode>
  We will now look at how we might use this modified component in our application
  ### Usage
  We can import the component exactly as before and it should work.
  <TitleCode title="author-modified-demo.mdx">
  ```jsx
  import Author from "@components/Author";

  <Author contributors={["manuanish", "github"]} readTime={5} />
  ```
