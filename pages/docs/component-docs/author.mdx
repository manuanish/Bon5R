  # Author
  <Contributors contributors={["manuanish"]} readTime={2} />
  ---
  The Author component can be used to preface a page linking to the author's GitHub account.
  <br/>
  ## Usage
  We can use the Author component like so:
  <TitleCode title="author.mdx">
  ```jsx
  import Author from "@components/Author";

  <Author
    author="Manu Anish"
    displayName="manuanish"
    date="3/14/15"
  />
  ```
  </TitleCode>
  <br/>
  This we be rendered as shown:
  <Demo>
    <Author
      author="Manu Anish"
      displayName="manuanish"
      date="3/14/15"
    />
  </Demo>
  <br />
  ### Props

| Name      | Type | Default | Description |
| --------- | ---- | ------- | ----- |
| `author`  |  `string`    |    `""`     | This is the value that will be name of the link.|
| `displayName` | `string` |     `""`    | This is the GitHub username of the author.      |
| `data`    |   `string`   |     `""`   | This is the date of posting.        |
