  # PageTitle
  <Contributors contributors={["manuanish"]} readTime={2} />
  ---
  The PageTitle component is used to change the title of the tab.
  <br/>
  ## Usage
  We can use the PageTitle component like so:
  <TitleCode title="page-title.mdx">
  ```jsx
  import PageTitle from "@components/PageTitle";

  <PageTitle title="My page title"/>
  ```
  </TitleCode>
  <br/>
  <br/>
  The following does not have any effect on the DOM rendering but if we look closely we can see that the title of the tab has changed to the provided string.
  <br/>

  <br />
  ### Props

| Name      | Type | Default | Description |
| --------- | ---- | ------- | ----- |
| `title` | `string` | `""` | The title of the webpage. |
