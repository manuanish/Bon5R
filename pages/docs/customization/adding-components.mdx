  # Creating your own components
  <Contributors contributors={["manuanish"]} readTime={5} />
  ---
  In this page we will look at an example that describes how we might create a Bon5R component.
  <br/>
  ## Creating the `TitleCode` component
  Let's begin by locating where all the components are located. If we navigate to the `components` folder we can see
  a list of the built in components. From here, we can create a new file called `TitleCode.js` and start creating our component
  <br/>
  <TitleCode title='TitleCode.js'>
    ```jsx
    export default function TitleCode(props) {
      return (
        <div className="flex flex-col overflow-hidden">
          <div className="bg-slate-800 w-fit p-2 pl-4 pr-4 rounded-tl-md rounded-tr-md mb-[-5px] z-10">
            <code className="">{props.title}</code>
          </div>
          <div>{props.children}</div>
        </div>
      );
    }
    ```
  </TitleCode>
  We will now look at how we might use this customized component in our application
  ### Usage
  We can import the component exactly as before and it should work.
  <TitleCode title="title-code-demo.mdx">
  ```jsx
  import TitleCode from "@components/docs/TitleCode";

  <TitleCode title="collatz.py">
    ~~~py
    def collatz(n):
      if n %% 2 == 0:
        return n / 2
      else:
        return 3 * n + 1
    ~~~
  </TitleCode>
  ```
  </TitleCode>
  <br/>
  This should give us the following
  <Demo>
  <TitleCode title="collatz.py">

  <br/>
  ### Props

| Name    | Type     | Default | Description              |
| ------- | -------- | ------- | ------------------------ |
| `title` | `string` | `""`    | Title of the code block. |
